Desktop Server Dashboard
========================

A local web server that runs on your desktop PC, displaying real-time
Swarm status, trading data, money metrics, and system health.

Run:
    python desktop_server.py

Open in browser:
    http://localhost:7777

Features:
- Real-time swarm status
- Money engine metrics
- Trading signals & alerts
- System health monitoring
- Auto-refresh every 5 seconds
